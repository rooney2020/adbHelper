# 交互状态矩阵

## 覆盖范围

- 核心页面：主工作台、高风险确认弹窗
- 核心控件：设备卡片、命令卡片、结果 Tab、搜索框、原始命令文本框、导出按钮
- 高风险交互：执行写操作、切换设备、结构化结果切换、错误回退到替代命令

## 高风险控件专项

| 控件 | 必查问题 | 验证方式 |
|------|---------|---------|
| 下拉框 | 箭头、展开态、popup 边缘、选中态 | 真实点击 + 截图 |
| 弹层 | 阴影、圆角、遮罩、边界 | 真实点击 |
| 列表 | hover、selected、scroll | 真实滚动 |
| 表单输入 | focus、error、placeholder | 键盘输入 |
| 结果 Tab | active、focus、切换动画 | 真实点击 |

## 验收证据

- 需要截图的状态：主工作台正常态、空态、加载态、错误态、高风险确认弹窗。
- 必须真实点击验证的状态：设备切换、结果 Tab 切换、风险弹窗开闭、错误态到替代命令提示。
- 必须在测试报告中单列的高风险点：高风险写操作确认、原文/结构化双视图一致性、长文本滚动性能。
